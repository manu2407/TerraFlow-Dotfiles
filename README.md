# TerraFlow - The Ultimate CachyOS Customization

![TerraFlow Banner](https://raw.githubusercontent.com/LpCodes/wallpaper/master/Abstract/topography.png)
*(Note: This is the default wallpaper. You can replace this with a screenshot of your desktop)*

**TerraFlow** is a comprehensive dotfiles repository designed to transform a stock **CachyOS** (or Arch Linux) installation into a high-performance, visually stunning, and fully automated workspace. It leverages the power of **Hyprland**, **Waybar**, **AGS**, and a suite of custom scripts to provide a seamless "TerraFlow" experience.

## üöÄ Features

- **Window Management**: Highly configured **Hyprland** with smooth animations and intuitive keybinds.
- **Status Bar**: Custom **Waybar** with modules for system info, media, and more.
- **Widgets**: **AGS** (Aylur's GTK Shell) for advanced widgets and on-screen displays.
- **Terminal**: **Kitty** terminal with **Fish** shell and **Starship** prompt for a modern command-line experience.
- **File Management**: **Yazi** for blazing fast terminal file navigation.
- **App Launcher**: **NWG-Drawer** and **Wofi** for quick app access.
- **Package Management**: Custom **Terra Store** TUI installer for interactive package management.
- **Aesthetics**: Unified theming with **Tokyo Night** colors, blurred lockscreens, and consistent fonts (Iosevka, Inter).
- **Tools**: Pre-configured **Lazygit**, **MPV** (with UOSC), **VS Code** extensions, and **Docker** setup.

## üì¶ Quick Start

To get started with TerraFlow, simply clone this repository and run the installation script:

```bash
git clone https://github.com/manu2407/TerraFlow-Dotfiles.git
cd TerraFlow-Dotfiles
./install.sh
```

For a detailed installation guide, please read [INSTALL.md](INSTALL.md).

## üìÇ Repository Structure

- **`configs/`**: Contains configuration files for all applications (hypr, waybar, fish, etc.).
- **`scripts/`**: Custom scripts for system management and the Terra Store.
- **`assets/`**: Wallpapers, fonts, and other resources.
- **`install.sh`**: The master installation script.
- **`packages.txt`**: List of packages installed by the script.
